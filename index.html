    Projects

    MACHINE LEARNING-BASED PRECANCEROUS CELL DETECTION ALGORITHM

    Details

    This started out as my undergraduate graduation project, in collaboration with a couple of other students and professors. We used a dataset containing optical imaging of human cells. It had previously been used as part of research to find light scattering patterns in precancerous cells. We implemented the same idea but with the help of machine learning concepts. My part involved working on supervised learning algorithms. The algorithms I custom developed for this cause included KNN, SVM and Decision Tree. I also used multiple versions of the dataset to create multiple models for testing. These versions were basically features extracted from the dataset itself. They included HOG, fourier descriptors, 1d fourior transforms and 2d fourier transforms. I was able to achieve test accuracies of up to 94.13%. 10-fold cross validation was implemented for improved reliability of results. I used Python as the programming language of choice and mostly the Scikit-learn library for pre-build ml functions.

    SENSOR MANAGEMENT PROGRAM - HARDWARE PROGRAMMING

    Details

    Developed a sensor management system that dapended on an ATmega128 microcontroller as its brains. It collected data off of multiple sensors connected to a gardening setup. It stored the data locally and performed functions such as switching LEDs and water motors on/off based on light level, temperature and moisture level readings. It was also connected to a keypad and display to make it easier for human users to interact with it. I used C as the programming language of choice and Atmel Studio + Proteus for code and design testing.
